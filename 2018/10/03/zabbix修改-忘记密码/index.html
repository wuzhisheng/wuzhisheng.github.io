zabbix修改/忘记密码

  • 监控

修改密码

点击 Administration -> Users -> Admin -> Password
再修改密码：cnhope
修改中文
Administration -> Users -> Admin ->Language
选：Chinese(zh_CN)


忘记Admin密码

mysql -uroot -pcnhope
show databases;
use zabbix
update users set passwd=md5(‘newpasswd’) where alias=’Admin’;
desc users; #查看user表字段
select * from users;
