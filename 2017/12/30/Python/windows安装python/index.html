windows安装python

  • 2017-12-30

下载python

www.python.org –> Downloads–>Windons 64位
下载后直接安装

  • 注意：在Customize Python2.7.11(64-bit)

    其他直接next

    验证

    在cmd界面
    python -V 有版本显示
    python -h 有python命令提示
    则安装成功

创建pip.ini文件，指定pip镜像

在C:\Users\吴志盛\pip下
创建pip.ini文档 ,并编辑
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
在cmd界面
python -m pip install -upfrade pip

下载安装pyCharm

http://www.jetbrains.com/pycharm/download/download-thanks.html?platform=windows

  • 选择默认安装路径
