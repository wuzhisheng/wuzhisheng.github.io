python一行命令搭建ftp

  • 2017-12-30

环境：python2.7

环境搭建

共享目录

  • 在cmd界面切换到想共享的目录下，运行

便可以浏览器或我的电脑：127.0.0.1:2121 （默认地址 端口）可以访问

可选参数

-i 指定IP地址（默认为0.0.0.0）
-p 指定端口（默认为2121）
-w 写权限（默认为只读）
-d 指定目录 （默认为当前目录）
-u 指定用户名登录
-P 设置登录密码

  • 要添加其他参数直接在 python -m pyftpdlib 命令后追加

    在共享目录追加写权限
    python -m pyftpdlib -w
